# ParacraftArtResource
The minimum art resource in paracraft
### Setup On Windows
- Installing Ant
- Set Environment of ${ParaEngineAB_Root} 
- After running ant
- All resource files will locate [res]
### Commands
- copy res
```
ant copyRes
```
